# Example: Todo App

A hello-world example of the OpenOwls SDD process — a simple todo app built spec-first.

## Status

The folder currently contains the blank OpenOwls SDD templates. The specs are being drafted in Claude.ai (Step 1 of the workflow). Once they're complete, Claude Code will build the app from the specs (Step 4).

Follow along:

1. **Draft specs in Claude.ai** — fill in each file under `ai_specs/` ← *we are here*
2. **Review** — read the specs as a team
3. **Fill in the gaps** — second pass on anything thin
4. **Build with Claude Code** — open this folder and let `CLAUDE.md` do the rest
